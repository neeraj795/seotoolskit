How Bulk Indexing Works

Google's Indexing API allows website owners to notify Google about new or updated pages. This tool streamlines the process by handling authentication, batching, and error recovery automatically.

  • Upload your Google service account JSON key files to authenticate
  • Paste URLs directly or upload a CSV file with your URL list
  • The tool submits each URL with controlled concurrency and retry logic
  • Track progress in real-time and download a detailed results report

How Internal Linking Works

Effective internal linking helps search engines understand your site structure and distributes page authority across your content. This tool automates the discovery of linking opportunities.

  • Enter your XML sitemap URLs to fetch all pages on your site
  • The tool crawls each page and extracts content for analysis
  • TF-IDF vectorization converts page content into numerical representations
  • Agglomerative clustering groups semantically similar pages together
  • Pages within the same cluster are suggested as internal linking targets
  • Download the complete linking plan as an Excel file
